# <Logo-Maker>

## Description

My motivation was to make a fun way to create logos. I built this project so everyday people can generate a logo when they need too. This application solves the problem of being able to make a logo even if the user isnt the most artistic. I learned how to use svg.

## Installation

First open the integrated terminal 
Next install package json with npm init -y
Then install the dependencies with npm i inquirer@8.2.4

## Usage

Run the program in the itergrated terminal by running node index.js
